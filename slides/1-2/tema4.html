"An Information-Theoretic Model for Steganography"

C

Розподіл "чистих" контейнерів

S

Розподіл стего-об'єктів

Міра безпеки: наскільки C та S статистично відрізняються?

KL-дивергенція

Kullback-Leibler дивергенція (відносна ентропія)

Вимірює "відстань" між двома розподілами ймовірностей

DKL(P || Q) = Σ P(x) log [P(x) / Q(x)]

Властивості:

DKL ≥ 0

Завжди невід'ємна

DKL = 0 ⟺ P = Q

Нуль при ідентичності

Несиметрична

D(P||Q) ≠ D(Q||P)

KL-дивергенція у стеганографії

DKL(PC || PS) = Σ PC(x) log [PC(x) / PS(x)]

Мала DKL

Розподіли схожі

→ Вища безпека

Велика DKL

Розподіли відрізняються

→ Легко виявити

DKL обмежує здатність будь-якого статистичного детектора

ε-безпека

Визначення

Стегосистема є ε-безпечною, якщо:

DKL(PC || PS) ≤ ε

Властивості:

  • Менше ε — вища безпека
  • Кількісна гарантія
  • Обмежує всіх детекторів

Практичні рівні:

  • ε < 0.01 — висока безпека
  • ε < 0.1 — помірна
  • ε > 1 — небезпечно

Досконала безпека

Найвищий рівень безпеки

Стегосистема є досконало безпечною, якщо:

DKL(PC || PS) = 0

Теорема Кашина

При DKL = 0 оптимальний статистичний детектор
не може зробити краще, ніж випадкове вгадування

Приховані дані повністю невиявлювані!

Інші міри відстані

Повна варіаційна відстань:

dTV(P, Q) = (1/2) Σ |P(x) - Q(x)|

Нерівність Пінскера:

dTV(P, Q) ≤ √[(1/2) DKL(P || Q)]

Практичне значення

Обмежуючи DKL (ε-безпека), ми автоматично обмежуємо dTV,
що пов'язана з ймовірністю виявлення
