Що таке інформація?

Визначення Шеннона

Інформація — міра зменшення невизначеності

Чим менш передбачувана подія — тим більше інформації несе її спостереження

I(x) = -log₂ p(x) біт

Приклад 1: Монета

p = 0.5

I = -log₂(0.5) = 1 біт

Приклад 2: Кубик

p = 1/6

I = -log₂(1/6) ≈ 2.58 біт

Ентропія Шеннона

Визначення

Ентропія — середня кількість інформації (невизначеності) у випадковій величині

H(X) = -Σ p(x) log₂ p(x) біт

Властивості ентропії:

H(X) ≥ 0

Завжди невід'ємна

Максимум

При рівноймовірних подіях

H(X) = 0

При детермінованій події

Приклад: ентропія мови

Ентропія англійської мови

H ≈ 2.62 біт/букву

Теоретичний максимум

26 рівноймовірних букв:

log₂(26) ≈ 4.7 біт

Надлишковість

Різниця:

4.7 - 2.62 = 2.08 біт

Ця надлишковість — простір для стеганографії!

Спільна ентропія

Невизначеність пари випадкових величин

H(X,Y) = -ΣΣ p(x,y) log₂ p(x,y)

Властивості:

H(X,Y) ≤ H(X) + H(Y)

Рівність тільки при незалежності

H(X,Y) ≥ max(H(X), H(Y))

Не менша за окремі

Умовна ентропія

Кількість інформації для опису Y при відомому X

H(Y|X) = -Σ p(x,y) log₂ p(y|x) = H(X,Y) - H(X)

Ключова властивість

H(Y|X) ≤ H(Y)

Знання X не може збільшити невизначеність щодо Y

Взаємна інформація

Кількість інформації, яку одна величина несе про іншу

I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

або еквівалентно:

I(X;Y) = H(X) + H(Y) - H(X,Y)
I(X;Y) ≥ 0
I(X;Y) = I(Y;X)
I = 0 ⟺ незалежні

Пропускна здатність каналу

Визначення

Максимальна швидкість надійної передачі інформації через канал

C = maxp(X) I(X;Y)

Максимум береться по всіх можливих розподілах входу

Теорема Шеннона про канал

При швидкості передачі нижче C можлива надійна комунікація
з довільно малою ймовірністю помилки

Підсумок теми

Поняття Формула Значення
Інформація I(x) = -log₂ p(x) Біти від події
Ентропія H(X) = -Σ p(x) log₂ p(x) Середня невизначеність
Умовна ентропія H(Y|X) = H(X,Y) - H(X) При відомому X
Взаємна інф. I(X;Y) = H(X) + H(Y) - H(X,Y) Спільна інформація
