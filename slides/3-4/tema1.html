Тема 1: Вступ, історичні методи та класифікація

Що таке текстова стеганографія?

Визначення

Форма прихованої комунікації, що приховує секретну інформацію в тексті природної мови

Мета

Зробити існування прихованого повідомлення невиявленим для сторонніх спостерігачів

Носій

Текстові документи, повідомлення, статті, веб-сторінки, код

Унікальні виклики

Текстова стеганографія є найскладнішим видом стеганографії

Низька надлишковість

Текст містить мінімум зайвих даних для приховування

Легка детекція

Модифікації легше помічає людина

Семантична коректність

Зміни повинні зберігати сенс тексту

Мала ємність

~0.5-2 біти на речення

Null Cipher (Нуль-шифр)

Стародавня форма стеганографії: відкритий текст змішаний з нешифрованим матеріалом

Приклад з Першої світової війни:

"PRESIDENT'S EMBARGO RULING SHOULD HAVE IMMEDIATE NOTICE." Перші літери: P-E-R-S-H-I-N-G (Генерал Pershing)

Типи правил:

  • Перша/остання літера кожного слова
  • Кожна n-та літера
  • Перша літера кожного другого слова

Акростих

Розширена форма null cipher: повідомлення приховано в перших літерах рядків

Світанок ніжно небо фарбував, Енергія нового дня зростала, Край лісу тихо просинався, Роса блищала, мов кришталь, Ехо співало пісню ранню, Тиша ховала таємницю. → Приховане слово: СЕКРЕТ

Історичне використання:

  • Таємні образи могутніх осіб
  • Кодування в опублікованих творах
  • Релігійні тексти
  • Дипломатичне листування

Шифр Бекона (1605)

Френсіс Бекон

Один з найраніших систематичних стеганографічних методів

Білітеральний шифр:

A = AAAAA H = AABBB O = ABBBA B = AAAAB I = ABAAA P = ABBBB C = AAABA J = ABAAB ... D = AAABB K = ABABA E = AABAA L = ABABB F = AABAB M = ABBAA G = AABBA N = ABBAB

Реалізація:

Звичайний шрифт = A
Курсив = B

Ємність:

5 символів cover-тексту на 1 літеру

Класифікація методів

Форматні методи

  • Whitespace (пробіли, табуляції)
  • Unicode zero-width символи
  • HTML/CSS маніпуляції

Лінгвістичні методи

  • Заміна синонімів
  • Синтаксичні трансформації
  • Семантичні методи

Feature-based

  • Line shifting
  • Word shifting
  • Feature coding

Генеративні методи

  • NLG (Natural Language Generation)
  • RNN/LSTM стеганографія
  • LLM-based методи

Критерії оцінки методів

C

Capacity (Ємність)

Максимум бітів для вбудовування

I

Imperceptibility

Невиявлюваність людиною

S

Security (Безпека)

Невиявлюваність алгоритмами

R

Robustness

Виживання після модифікацій

Підсумок теми

Ключові поняття:

  • Текстова стеганографія — приховування даних у тексті природної мови
  • Null cipher — найдавніший метод (перші літери слів)
  • Акростих — приховання в перших літерах рядків
  • Шифр Бекона — перший систематичний метод (1605)

Головна проблема

Текст має найменшу надлишковість серед усіх типів носіїв,
що робить текстову стеганографію найскладнішою
