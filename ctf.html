Що таке стеганографія?

Визначення

Стеганографія (грец. steganos — прихований, graphein — писати) — наука про приховану передачу інформації шляхом збереження в таємниці самого факту передачі.

На відміну від криптографії, де повідомлення зашифроване, але його наявність очевидна, стеганографія ховає сам факт існування секретного повідомлення.

Аналогія

Криптографія — ви бачите сейф, але не знаєте код.

Стеганографія — ви навіть не знаєте, що сейф існує.

Оригінал
R: 192 G: 140 B: 96
Після LSB
R: 193 G: 141 B: 97

Зміна найменш значущого біта не помітна оку — різниця лише ±1 на канал

Основні методи стеганографії у CTF

LSB (Least Significant Bit)

Найпопулярніший метод. Дані записуються у найменш значущі біти пікселів зображення. Зміна ±1 не помітна оку.

zsteg stegsolve StegOnline

Метадані (EXIF/PNG Comments)

Інформація ховається у метаданих файлу: EXIF, PNG text chunks, IPTC, XMP. Перевіряй exiftool!

exiftool pngcheck identify

Вбудовані файли

Файли можуть бути склеєні: після кінця PNG можна дописати ZIP-архів. Зображення відкриється нормально, але binwalk знайде вкладення.

binwalk foremost xxd

Аудіо стеганографія

Дані можуть бути у LSB аудіо, у спектрограмі (видно як зображення) або в ехо-сигналі.

Audacity sonic-visualiser sox

Текстова стеганографія

Пробіли та табуляції в кінці рядків кодують біти. Unicode zero-width символи (U+200B, U+200C) невидимі, але несуть інформацію.

cat -A xxd Python

Ланцюжки кодувань

Прапор закодований кілька разів: base64 → ROT13 → hex → binary. CyberChef з кнопкою Magic автоматично розпізнає ланцюжок!

CyberChef base64 Python

Часті помилки початківців

  • Пропускають strings — іноді прапор лежить відкритим текстом!
  • Не перевіряють метадані — exiftool має бути першим кроком
  • Забувають про binwalk — багато файлів містять вкладення
  • Не знають про CyberChef Magic — він автоматично розпізнає кодування
  • Використовують тільки 1-бітну LSB — бувають 2-бітні та 4-бітні варіанти

Зображення: LSB, метадані, вбудовані файли

Метадані

exiftool image.png pngcheck -v image.png

LSB (Least Significant Bit)

zsteg image.png # Автоматичний аналіз zsteg -a image.png # Всі комбінації stegsolve # GUI: bit planes, data extract

Вбудовані файли

binwalk challenge.png # Пошук binwalk -e challenge.png # Витягнення foremost challenge.png # Альтернатива
Текст: whitespace, Unicode

Whitespace стеганографія

cat -A challenge.txt # $ = кінець рядка, ^I = tab # Пробіл = 0, Tab = 1, 8 біт = 1 ASCII символ

Unicode zero-width символи

# Шукаємо невидимі символи: U+200B, U+200C, U+200D, U+FEFF python -c " with open('f.txt') as f: for i,ch in enumerate(f.read()): if ord(ch) > 127: print(f'pos {i}: U+{ord(ch):04X}') "
Аудіо: LSB, спектрограма, echo hiding

Спектрограма

Audacity → клік на доріжку → Spectrogram → шукай текст/зображення

Audio LSB

python -c " import wave w = wave.open('challenge.wav','rb') frames = w.readframes(w.getnframes()) bits = ''.join(str(b & 1) for b in frames) text = ''.join(chr(int(bits[i:i+8],2)) for i in range(0,len(bits),8)) print(text[:200]) "
Кодування: base64, hex, rot13, binary

Розпізнавання

  • Base64: A-Z, a-z, 0-9, +, / з = на кінці
  • Hex: тільки 0-9, a-f, парна кількість
  • Binary: тільки 0 і 1, групи по 8

Декодування

echo "SGVsbG8=" | base64 -d # base64 echo "48656c6c6f" | xxd -r -p # hex echo "Uryyb" | tr 'A-Za-z' 'N-ZA-Mn-za-m' # rot13
